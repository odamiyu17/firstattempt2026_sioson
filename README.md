# Alumni Career Hub (PWA)

A mobile-first Progressive Web App built using React + Vite for Ateneo de Davao University.

## Features
- Login UI
- Home Dashboard
- Events Listing
- Event Details
- Profile Page
- Academic Records Page

## Tech Stack
- React + Vite
- React Router
- CSS
- vite-plugin-pwa
